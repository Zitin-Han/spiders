# DXYS 丁香医生爬虫

基于 Selenium + BeautifulSoup 的丁香医生（dxy.com）问答数据爬虫，批量采集全科室医患对话及医生信息。

## 功能

- 自动获取丁香医生全部科室列表（33个科室）
- 遍历每个科室下的疾病分类
- 分页抓取问答列表（每疾病最多10页）
- 进入详情页解析：
  - 患者与医生的**完整对话记录**
  - 医生信息（姓名、职称、科室、医院、评分、回复数）
  - 对话时间
- 导出为 Excel 并自动去重

## 数据字段

| 字段 | 说明 |
|------|------|
| 科室 | 所属科室 |
| 疾病 | 疾病分类 |
| 对话时间 | 问答时间 |
| 医生信息 | 姓名 / 职称 / 科室 / 医院 / 评分 / 回复数 |
| 对话全文 | 患者-医生完整对话 |
| 链接 | 问答页面 URL |

## 数据文件

- `丁香医生_全科室_13524.xlsx` — 全量采集（13,524 条）
- `丁香医生_全科室_去重_13323.xlsx` — 按链接去重后（13,323 条）

## 安装

```bash
pip install selenium pandas openpyxl beautifulsoup4
```

需配合 Chrome 浏览器与对应版本 ChromeDriver。

## 使用

```bash
jupyter notebook code.ipynb
```

按顺序执行 Cell 即可完成：
1. Selenium 爬取 → 保存 Excel
2. 读取 → 去重 → 导出
